OpenMPExample
=========
Application is example of usage OpenMP framework within C++ code

Compilation
---------
Go to main source code directory and invoke following commands:
```
mkdir build
cd build
cmake ..
make 
sudo make install
```

Usage
---------
openMPExample

Output
---------
Output is statistics of usage OpenMP framework in C++ code
